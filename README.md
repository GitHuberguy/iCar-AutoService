# iCar-AutoService
This project is a web-based system developed as a Live Demo for Software Engineering 2 course.

# Project Description
iCar AutoService is a system that allows users to view car services and book a service appointment.
An admin page is available to view all bookings and change their status.

# Used Technologies
- HTML
- CSS
- JavaScript

# How to Run
1. Open the project folder in Visual Studio Code.
2. Open index.html
3. Click "Go Live" using the Live Server extension.
4. Navigate through the page using menu.
